# Playbook — Zipper Abordagem Obra/Colecionador

## Objetivo

Preparar abordagens comerciais sofisticadas para colecionadores sem hard sell, sem inventar histórico e sem confundir dados de vendas da Zipper com leilão SPITI.

## Quando usar

- Lucas/Osmar pedem sugestão de abordagem para obra, artista ou colecionador.
- Há interesse em ativar base de colecionadores.
- Uma obra precisa de narrativa comercial antes de contato.

## Fontes

1. `memories/zipper.md` para tom, metas e separações críticas.
2. Zipper Vendas/Supabase `pcstqxpdzibheuopjkas`, tabela `vendas_tango`, quando houver afirmação de venda real.
3. `areas/zipper/sub-areas/colecionadores/MAPA.md`.
4. `areas/zipper/sub-areas/vendas-obras/MAPA.md`.
5. Materiais confirmados de obra/artista fornecidos por Lucas/Osmar/equipe.

## Loop

```text
obra/artista/colecionador → histórico confirmado → narrativa cultural → rascunho → aprovação Lucas/Osmar → contato → resultado → CRM/lesson
```

## Regras de linguagem

Usar:

- curadoria;
- pertencimento;
- contexto cultural;
- trajetória do artista quando confirmada;
- tom leve, culto e humano.

Evitar:

- pressão de venda;
- "oportunidade imperdível";
- liquidação/desconto como eixo narrativo;
- superlativos sem base;
- afirmar preferência ou poder de compra sem histórico.

## Checklist de verificação

- [ ] A obra/artista existe no contexto informado.
- [ ] Histórico de compra/interesse foi consultado ou marcado como `[a confirmar]`.
- [ ] Dado comercial veio de Zipper Vendas, não de SPITI.
- [ ] O texto não inventa relação prévia.
- [ ] O texto está em formato de rascunho, não envio automático.
- [ ] Lucas/Osmar são apontados como aprovadores antes do contato.

## Saída esperada

```text
Contexto: ...
Fonte consultada: ...
Hipótese de interesse: ...
Rascunho de abordagem: ...
Tom usado: ...
Ressalvas: ...
Aprovação necessária: Lucas/Osmar antes de enviar
```

## Pós-contato

Depois que a equipe registrar retorno, atualizar:

- interesse demonstrado;
- objeções;
- próxima obra/artista sugerida;
- lição sobre tom, timing ou segmentação.
