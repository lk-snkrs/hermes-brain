# Rotina — Abordagem de colecionadores Zipper

## O que faz

Prepara rascunhos de abordagem para colecionadores com base em histórico real e narrativa cultural.

## Loop

```text
perfil → histórico → obra/artista → narrativa → rascunho → aprovação Lucas/Osmar → contato → resultado
```

## Aprovação obrigatória

Nenhuma mensagem deve ser enviada para colecionador sem aprovação explícita de Lucas/Osmar.

## Verificação

- Histórico consultado.
- Obra/artista confirmados.
- Tom Zipper aplicado.
- Sem hard sell.
- Resultado registrado após contato.
