## Regras globais

1. Ler globalmente, mas classificar antes de agir.
2. Salvar dados estruturados no contexto correto: Zipper, SPITI, LK ou Lucas/global.
3. Não misturar tom, CRM, fonte de verdade ou credenciais entre empresas.
4. E-mail/WhatsApp externo permanece approval-gated, exceto subfluxos fechados explicitamente aprovados.
5. Brain/KB não guarda senhas, tokens, dados bancários nem PII completa desnecessária.
