# Approval Ledger — Operações

Livro-razão para aprovações/correções de Lucas que mudam comportamento operacional do Hermes.

Use o template `areas/operacoes/templates/approval-ledger-entry.md`.

## Regra

- Aprovação que altera autonomia, envio externo, copy, tom, fluxo, infra, campanha, fornecedor, preço ou disponibilidade precisa de registro durável.
- Se for específica de uma empresa, registrar também na área da empresa e linkar a partir daqui quando necessário.
