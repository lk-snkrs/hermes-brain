# REGRA — LKGOC Shared Shell: só texto e imagem mudam

Registrado em: 20260606T185527Z
Status: **BLOQUEANTE / CANÔNICO**

## Regra curta

Todas as coleções LKGOC devem compartilhar o mesmo shell visual aprovado da New Balance 204L. Entre coleções, só podem mudar texto, fotos/imagens, links e conteúdo específico estritamente necessário.

## Razão

Se Lucas mudar algo no tema/padrão no futuro, as coleções LKGOC devem acompanhar de forma consistente. Cada coleção não pode virar um layout próprio.

## Obrigatório

- Usar componente/padrão compartilhado.
- Manter visual praticamente idêntico ao 204L.
- Parametrizar conteúdo por coleção quando possível.
- Evitar branches com CSS/layout próprios.
- Qualquer exceção visual exige aprovação explícita de Lucas.

## Reprova

- Layout novo por coleção.
- CSS específico que altere shell visual.
- Branch que replica marca, mas muda estrutura.
- QA técnico sem validar que só texto/imagem mudaram.

## Production

Production/main continua bloqueado sem aprovação explícita.
