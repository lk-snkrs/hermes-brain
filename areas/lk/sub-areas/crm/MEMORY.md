# MEMORY — LK CRM

## Decisões ativas

- Contato externo de CRM sempre exige aprovação explícita e conteúdo final aprovado.
- Não sugerir produto fora de estoque; consultar fonte viva quando a oferta depender de disponibilidade.
- Copy customer-facing não deve expor jargões internos como Brain, P1, preview técnico, SKU/Tiny ou mecânica de estoque.

## Guardrails lembrados

- CRM pode preparar, segmentar e recomendar; envio é aprovação separada.
- Aprendizado útil deve voltar para playbook/rotina, não ficar preso no chat.
