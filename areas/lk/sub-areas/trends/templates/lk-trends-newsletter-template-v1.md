## Regra de layout

A newsletter LK-TRENDS deve usar o **mesmo layout/base visual das newsletters LK/Klaviyo já existentes**, não um layout novo de relatório.

Fonte visual canônica:

- `areas/lk/design/DESIGN.md`
- `areas/lk/sub-areas/crm/rotinas/playbook-newsletter-klaviyo-lk.md`
- Skill `lk-growth-operations` / seção Klaviyo-newsletter

Direção obrigatória:

- visual premium, silencioso, editorial e curatorial;
- largura de e-mail 600–640px;
- header preto com logo LK branco centralizado;
- context bar/microcopy uppercase;
- títulos serifados `Cormorant Garamond, Georgia, serif`;
- corpo/UI em `DM Sans, Arial, sans-serif`;
- fundo paper/off-white quente;
- CTA preto compacto;
- cards sem sombra forte, sem radius evidente;
- produto/tendência como protagonista visual;
- footer preto com slogan: `O que é raro, merece ser encontrado`.

Markdown serve apenas como briefing. Para aprovação visual, gerar **HTML/rendered preview** no padrão newsletter LK.
