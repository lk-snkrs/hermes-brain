# TOOLS — LK Shopify

## Toolsets recomendados

- file/search: documentação, previews, receipts.
- terminal limitado: scripts locais e validações, sem secrets impressos.
- web/browser apenas quando necessário para inspeção pública ou preview.
- skills/session_search: procedimentos e histórico.

## Evitar por padrão

- messaging externo;
- cronjob;
- writes API sem approval packet;
- qualquer ferramenta que mude Shopify/Tiny sem escopo aprovado.
