# Handoff — LK Padrão

Use este template para handoffs de LK Ops, LK Trends e LK Shopify quando houver decisão, preview, bloqueio, write aprovado, risco ou aprendizado reutilizável.

```md
## YYYY-MM-DD HH:MM — [especialista] — [tema]

- Pedido/evento:
- Fonte viva consultada:
- Item/SKU/tamanho/cliente/pedido/fornecedor:
- Resultado verificado:
- Output/rascunho:
- Writes externos: não/sim
- Aprovação: não aplicável / pendente / aprovada por [quem] em [quando]
- Snapshot/readback/receipt:
- Risco/bloqueio:
- Próximo passo:
- Onde ficou documentado:
```

Regra: se houve write externo, anexar ou referenciar snapshot, readback e rollback.

Padrão de uso:
- LK Ops/Atendimento usa quando o assunto for atendimento, estoque, preço, disponibilidade, reserva, pedido ou relatório comercial.
- LK Trends usa quando houver oportunidade, sourcing, risco, fornecedor, compra/recomendação ou insight reutilizável.
- LK Shopify usa quando houver preview, produto/upload, coleção, superfície Shopify, write aprovado ou bloqueio.
