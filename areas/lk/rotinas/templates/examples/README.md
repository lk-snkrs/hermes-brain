## Regras

- Os números são fictícios e marcados como exemplo.
- Não representam vendas reais da LK.
- Não usam dados pessoais reais.
- Não autorizam compra, mensagem, campanha, publicação, preço ou alteração de estoque.
- A utilidade aqui é validar o formato de decisão: fato → leitura → ação sugerida → risco → aprovação.
