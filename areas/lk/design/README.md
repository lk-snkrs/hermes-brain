# Design LK

Sistema visual e previews derivados da comunicação real da LK.

## Arquivos

- `DESIGN.md` — DesignMD v0.1 da LK, baseado nos últimos e-mails enviados no Klaviyo.
- `klaviyo-visual-audit.md` — auditoria resumida dos e-mails usados como referência.
- `previews/lk-designmd-pdp-preview.html` — HTML visual para aprovação do primeiro padrão de PDP/produto. Usa logo real da LK e imagens reais embutidas para evitar falha de carregamento no preview.

## Regra

Antes de gerar layout de produto, PDP, newsletter, blog, carrossel, Klaviyo ou WhatsApp visual, consultar `DESIGN.md`.

Aprovação de layout deve ser feita com HTML visual direto, não apenas Markdown.
