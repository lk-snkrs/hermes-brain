# Design LK

Sistema visual e previews derivados da comunicação real da LK.

## Arquivos

- `DESIGN.md` — DesignMD v0.1 da LK, baseado nos últimos e-mails enviados no Klaviyo.
- `klaviyo-visual-audit.md` — auditoria resumida dos e-mails usados como referência.
- `previews/lk-designmd-pdp-preview.html` — HTML visual interno para calibrar a direção estética do DesignMD/PDP. Usa logo real da LK e imagens reais embutidas para evitar falha de carregamento no preview. Não é template final, não é peça de cliente e não deve ser publicado/disparado.

## Status do preview atual

- Objetivo: validar estética global v0.1 — botões, cores, grid, hierarquia, logo, produto e tom visual.
- Uso permitido: referência interna para DesignMD e próximos templates reais.
- Uso proibido: enviar para cliente, publicar na Shopify, usar em Klaviyo, disparar no WhatsApp ou tratar como PDP final.
- Aprovação Lucas: direção visual aprovada para seguir; peça não aprovada para uso externo.

## Regra

Antes de gerar layout de produto, PDP, newsletter, blog, carrossel, Klaviyo ou WhatsApp visual, consultar `DESIGN.md`.

Aprovação de layout deve ser feita com HTML visual direto, não apenas Markdown.
