# TOOLS — [LK] Otimização de Coleção

## Read-only permitido

- Web/SERP e fontes editoriais/oficiais.
- DataForSEO quando disponível.
- GSC/GA4/GMC/Ahrefs quando disponível em modo read-only.
- Shopify read-only para coleção, página, theme, assets e public checks.
- File/search/read/write local no Brain.

## Criação local permitida

- PRD, input contract, evidence packet, draft, scorecard, approval packet, receipt e impact review.

## Approval-gated

- Shopify writes, theme/assets, pages, collections, metafields, SEO fields, production publish.
- Tiny, GMC/feed, Klaviyo, Meta/Google Ads, WhatsApp/email.
- Cron novo, gateway/runtime externo, Docker/VPS/Traefik/secrets.

## Segurança

Nunca imprimir tokens/secrets. Nunca salvar credenciais no Brain. Receipts devem dizer que token/segredo foi usado sem revelar valor.
